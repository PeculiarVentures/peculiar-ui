## Size

Use the `size` prop to change the size of the text field. You can set the value to `small`, `medium` (default) or `large`.

<Canvas isColumn>
<TextField size="small" defaultValue="Small" placeholder="Size" />
<TextField size="medium" defaultValue="Medium" placeholder="Size" />
<TextField size="large" defaultValue="Large" placeholder="Size" />
</Canvas>

## Color

Supports only `secondary` color.

#### Secondary color

<Canvas>
<TextField placeholder="Color" defaultValue="Secondary" />
</Canvas>

## Validation

The `error` prop toggles the error state. The `errorText` prop can then be used to provide feedback to the user about the error.

<Canvas>
  <TextField
    error
    errorText="Incorrect entry."
    placeholder="Error"
    defaultValue="Hello World"
  />
</Canvas>
