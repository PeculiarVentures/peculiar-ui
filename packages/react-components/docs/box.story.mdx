# Box

The Box component serves as a wrapper component for quickly change `background` and `border` properties.

## Color

Use the `background` prop to change the color of the Box background color (see [props](#props) section).

<Canvas>
<Box
  background="attention"
  style={{
    padding: 10
  }}
>
  Attention
</Box>
<Box
  background="gray-6"
  style={{
    padding: 10
  }}
>
  Gray 6
</Box>
</Canvas>
