## Color

Use the `color` prop to change the color of the typography (see [props](#props) section).

<Canvas isColumn>
<Typography color="primary" variant="h5">
  Heading
</Typography>
<Typography color="secondary" variant="h5">
  Heading
</Typography>
<Typography color="wrong" variant="h5">
  Heading
</Typography>
<Typography color="attention" variant="h5">
  Heading
</Typography>
</Canvas>

## Truncate text

Pass the `noWrap` prop to render an ellipsis when the text exceeds the width of the viewport.

<Canvas>
<Typography noWrap>
  Lorem ipsum is placeholder text commonly used in the graphic, print, and publishing industries for previewing layouts and visual mockups.
</Typography>
</Canvas>
