# Dark mode

You can make your application use the dark theme as the default—regardless of the user's preference—by adding mode: 'dark' to the `ThemeProvider`:

```tsx
import { Button, ThemeProvider } from '@peculiar/react-components';

function App() {
  return (
    <ThemeProvider mode="dark">
      <Button>
        Action
      </Button>
    </ThemeProvider>
  );
}
```
